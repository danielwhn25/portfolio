Laser Timing Gates

Project Lead | Formula UBC Racing

Project Overview

Led the design and development of a precision laser timing gate system for Formula UBC Racing. The system uses 650 nm laser diodes and custom transimpedance amplifier (TIA) circuits to detect vehicle passage with high accuracy and reliability.

Technical Specifications

Wavelength
650 nm
Signal-to-Noise Ratio
20 dB
Bit Error Rate
< 10⁻⁷
Design Tool
Altium Designer

Key Achievements

  • Designed custom transimpedance amplifier (TIA) circuit optimized for 650 nm laser detection
  • Developed complete PCB layout in Altium Designer with proper RF considerations
  • Calibrated system to achieve 20 dB SNR for reliable signal detection
  • Achieved bit error rate below 10⁻⁷ for high-precision timing measurements
  • Led cross-functional team coordination between electrical and mechanical subsystems
  • Validated system performance through extensive testing and characterization

Technical Challenges

  • Optimized TIA design for low-noise amplification of photodiode signals
  • Ensured robust operation in outdoor racing environment with varying lighting conditions
  • Maintained timing accuracy requirements for competitive racing applications
  • Integrated laser transmitter and receiver subsystems for reliable detection

Technologies Used

  • Altium Designer (PCB Design)
  • 650 nm Laser Diodes
  • Custom TIA Circuit Design
  • RF/High-Speed PCB Layout
  • Signal Processing & Calibration

Impact

Delivered a reliable timing gate system that enables precise lap time measurements for Formula UBC Racing, supporting data-driven performance analysis and race strategy optimization.
