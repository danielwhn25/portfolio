MPPT Control System: PVA Analyzer

Hackathon Project
STM32G4 // CAN-Bus // 4-Layer PCB // Altium Designer
  • Power Efficiency: Recaptured up to 15% of energy typically lost to sensor noise in solar arrays by implementing precise MPPT tracking on the STM32G4.
  • EMI Mitigation: Shielded high-frequency ADC telemetry from switching noise using internal ground planes on a 4-layer stackup.
  • Robust Comm: Ensured fault-tolerant data transmission across the vehicle via noise-immune CAN-Bus, enabling reliable real-time power telemetry.
  • Scalability: Allowed for high-power application scaling using a modular, decoupled Buck-Boost architecture separable from the control logic.
