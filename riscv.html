RISC-V Single Cycle CPU

Academic Project | December 2025

Project Overview

Designed and implemented a complete RISC-V single-cycle processor in System Verilog, supporting a 10-instruction subset of the RISC-V ISA. The processor executes one instruction per clock cycle and includes comprehensive verification through assertion-based testbenches.

Architecture

  • Single-cycle execution model - one instruction per clock cycle
  • RISC-V 32-bit instruction set architecture
  • 10-instruction subset implementation
  • Modular design with separate datapath and control unit
  • Register file with 32 general-purpose registers
  • ALU supporting arithmetic and logical operations

Instruction Set

The processor implements a 10-instruction subset including:

ADD
SUB
AND
OR
XOR
SLT
ADDI
LW
SW
BEQ

Key Components

  • Control Unit: Generates control signals based on instruction opcode
  • Datapath: Implements instruction execution pipeline
  • Register File: 32x32-bit registers with dual-port read, single-port write
  • ALU: Performs arithmetic, logical, and comparison operations
  • Instruction Memory: Stores program instructions
  • Data Memory: Handles load/store operations
  • Program Counter: Tracks instruction execution flow

Verification & Testing

  • Developed comprehensive System Verilog testbenches
  • Implemented assertion-based verification (ABV) for formal validation
  • Created test programs covering all instruction types
  • Verified correct execution of arithmetic, logical, and control flow instructions
  • Validated memory access operations (load/store)
  • Tested edge cases and error conditions

Technical Skills

  • System Verilog Hardware Description Language
  • RISC-V Instruction Set Architecture
  • CPU Architecture & Microarchitecture Design
  • Digital Logic Design
  • Assertion-Based Verification
  • Hardware Simulation & Testing

Learning Outcomes

This project provided deep understanding of processor architecture, instruction execution, and hardware verification methodologies. The single-cycle design demonstrates fundamental concepts that extend to pipelined and superscalar processor architectures.
